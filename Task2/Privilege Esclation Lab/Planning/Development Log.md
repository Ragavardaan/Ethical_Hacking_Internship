Development Log



Project Title



SecureTech Linux Privilege Escalation Training Lab



Day 1 – Project Planning



\#Activities Completed



\- Selected Linux Privilege Escalation as the project topic.

\- Created the project directory structure.

\- Planned the overall architecture of the training lab.

\- Identified the required documentation files.

\- Prepared Oracle VirtualBox for the Ubuntu Server installation.



\#Outcome



A clear project roadmap was prepared before implementation.



Day 2 – Environment Setup



\#Activities Completed



\- Installed Ubuntu Server 26.04 LTS on Oracle VirtualBox.

\- Updated the operating system.

\- Installed the required Linux packages.

\- Created administrator and learner user accounts.

\- Enabled SSH support.

\- Verified the environment configuration.



\#Outcome



A clean Linux environment was successfully prepared for building the training lab.



Day 3 – Lab Development



\#Activities Completed



\- Created the SecureTech company directory structure.

\- Added department folders for HR, Finance, IT, and Backups.

\- Created sample files representing confidential company data.

\- Configured Linux permissions and ownership.

\- Added learner flag files.

\- Configured controlled sudo permissions for the learner account.

\- Tested user access and verified the environment.



Outcome



The lab environment was successfully transformed into a realistic enterprise-style Linux server for cybersecurity training.
